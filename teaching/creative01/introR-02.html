R에서 MS 엑셀화일을 불러들이는 방법

예제: 엑셀 화일의 구성

MS 엑셀화일(Miscrosoft Excel file)은 자료를 저장하고 정리하는데 매우 유용한 프로그램이다. 이번에는 R에서 MS 엑셀화일을 불러들여서 데이카프레임 형식으로 저장하는 방법을 소개한다.

아래 그림과 같이 2014-2018년의 서울시 구별 인구정보가 test.xlsx 엑셀 화일에 저장되어 있다고 가정하자.

  • 첫 번째 줄은 각 열을 설명하는 변수의 이름이며 영어문자로 구성되어 있다.
  • 5개의 변수로 구성된 자료이다. (year, name, house, pop, old65)
    • year: 년도
    • name : 구이름
    • house: 가구수
    • pop : 인구수
    • old65: 65세 이상 인구수
  • 한 개의 sheet만 있는 화일이다 (참고로 MS 엑셀화일은 여러 개의 sheet으로 구성될 수 있다.)

2018년 서울시 구별 인구 엑셀 화일

패키지의 설치와 이용 지정

패키지 readxl의 설치

R은 독자적으로도 여러 가지 다양한 기능을 수행할 수 있는 통계 프로그램이지만 다양한 기능을 제공하는 패키지(poackage)를 설치하면 더욱 편리한 기능을 이용할 수 있다.

MS 엑셀화일은 여러 가지 형식(xls, xlsx, csv, …)이 있기 때문에 엑셀화일을 수월하게 읽을 수 있는 패키지 readxl을 설치하여 이용하면 쉽게 엑셀화일의 자료를 데이터프레임으로 변환하여 사용할 수 있다.

패키지의 설치는 두 가지 방법이 있다.

  1. plot 패널에서 packages 탭을 클릭하면 install 버튼이 있다. install 버튼을 클릭하면 대화창이 뜨고 설치하고자 하는 패키지의 이름을 입력하여 선택한다. 아래 있는 install 버튼을 클릭하면 패키지를 설치해 준다.

Rstudio에서 패키지 설치

  1. R의 console화면에서 명령어 install.packages 함수를 이용하여 설치한다.
install.packages("readxl")

패키지의 지정

패키지 readxl을 설치한 후 그 기능을 이용하고 싶다면 다음과 같이 library 함수를 사용하여 패키지를 지정해 주어야 한다.

library(readxl)

참고로 패키지의 설치는 최초 한 번만 해주면된다. 하지만 library 함수를 이용한 패키지의 지정은 사용할 때마다 실행해 주어야 한다.

엑셀 화일의 입력

working directory 지정

이제 test.xlsx 엑셀 화일을 원하는 디렉토리에 저장한다.

화일을 저장한 디렉토리를 working directory로 지정해 주어야 한다. Rstudio 메뉴에서 session을 선택하면 set working directory 메뉴가 있다. choose directory를 선택하고 test.xlsx 엑셀 화일을 저장한 디렉토리를 지정해 준다.

엑셀 화일의 입력

CSV 화일의 입력

많은 경우 자료가 CSV(comma-separated values)화일로 저장된다. CSV 화일은 자료의 값(필드)를 쉼표(,) 또는 여백(space)로 구분한 텍스트 데이터 및 텍스트 파일이다. 확장자는 .csv 이며 이 화일은 MS Excel 프로그램으로 볼 수 있다.

예제 화일인 seoulpopall.csv의 실제 내용을 보면 다음과 같다.

year,name,house,pop,popm,popf,for,density,area,numhouse,old65
1991,합계,3330317,10904527,5485353,5419174,,18014,605.33,3.27,
1991,종로구,78169,236806,119127,117679,,9865,24.01,3.03,
1991,중구,61495,184142,93312,90830,,18438,9.99,2.99,

CSV 화일은 R 에서 함수 read.table를 이용하여 자료를 불러 올 수 있다. 아래는 seoulpopall.csv 화일의 자료를 불러오는 R 명령어이다.

df <- read.table("seoulpopall.csv", sep=",", header = TRUE)

함수 read.table의 세가지 인자를 설명하면 다음과 같다.

  • "seoulpopall.csv": 문자열로 된 csv 화일 이름
  • sep="," : 각 변수의 필드가 쉼표(,)로 분리된 것을 지정한다. 만약 여백(space)으로 구분되어 있다면 sep=""로 지정한다.
  • header = TRUE : 첫 행이 자료가 아니고 변수의 이름이라는 것을 알려주는 문장이다.

아래는 seoulpopall.csv 화일의 자료가 저장된 데이터프레임 df의 내용이다. 자료 중에 NA로 표시된 부분은 자료가 없는 것, 즉 결측값(missing value, Not Available)을 의미 한다.

비교연산자를 이용한 자료의 선택

이제 위에서 만든 데이터프레임 seoulpop02를 보면 구의 이름 name 이 합계인 자료가 나타나고 있다. name 이 합계인 자료(행)를 데이터프레임에서 제거해 보자.

첫 번째 방법은 먼저 name 이 합계인 자료(행)를 찾는 것이다. 다음과 같은 비교연산자 ==를 사용하면 벡터 seoulpop02$name 이 합계인 행이 TRUE, 아닌 행이 FALSE로 나타나는 논리값 벡터를 만들 수 있다.

myselect <- seoulpop02$name == "합계"

비교연산자는 다음과 같은 6가지가 있다.

  • == : is equal?

  • != : is not equal?

  • > or >=: is greater (greater or equal)?

  • < or <= : is less (less or equal)?

  • & : AND 연산

  • | : OR 연산

만약에 자료에서 2018년도 구인구가 50만명을 넘는 자료만 선택하여고 한다면 다음과 같은 명령어로 선택이 가능하다. 여기서 연도를 나타내는 seoulpop02$year의 자료값이 문자열인것을 유의하자. 인구를 나타내는 변수 seoulpop02$pop는 숫자이다.

myselect2 <- seoulpop02$year == "2018" & seoulpop02$pop > 500000

패키지 dplyr을 이용한 자료의 선택

위에서 비교연산자를 이용하면 데이터프레임에서 조건에 맞는 자료만을 선택할 수 있는 방법을 알아보았다. 이러한 자료의 선택은 패키지 dplyr를 사용하면 더욱 쉽게 작업을 할 수 있다.

먼저 위에서 한 것과 같은 동일한 방법으로 패키지 dplyr을 설치하고(전에 설치했다면 다시 설치할 필요없다) 이용할 수 있도록 지정한다.

함수 filter의 첫 번째 인자는 데이터프레임의 이름이다. 그 다음 인자들는 차례로 원하는 조건을 열거하여 컴마로 분리하면 조건들에 맞는 자료만 선택하게 된다. 조건문을 쓸 때 데이터프레임안의 변수명만 사용하면 된다.

위에서 2018년도 구인구가 50만명을 넘는 자료만 선택하는 작업을 함수 filter를 이용하여 수행하고 tempdata에 저장해 보자.

tempdata <- filter(seoulpop,  year == "2018" , pop > 500000)

자료의 그룹별 요약

이제 자료를 그룹별로 요약하여 기초 통계량을 구하는 방법을 알아보자.

이전의 예제에서 구의 이름 name이 합계인 자료를 모두 제거했는데 데이터프레임 seoulpop 를 이용하여 이제 연도별 서울 전체인구를 구하는 방법을 알아보자. 먼저 데이터프레임에 있는 연도 year를 그룹으로 구별인구 pop를 합계해서 년도별 서울인구를 구해야 한다.

이러한 작업은 패키지 tidyr의 연결연산자 %>%를 사용하면 편하다. 먼저 패키지 tidyr를 지정하자 (앞에서와 마찬가지로 패키지 tidyr가 설치되어 있지 않으면 먼저 설치한다)

library(tidyr)

이제 연결연산자를 이용하여 연도별 서울 전체인구를 구하는 명령어는 다음과 같다.

seoulpop %>% group_by(year) %>% summarise(totalpop = sum(pop)) 
## # A tibble: 5 x 2
##   year  totalpop
##   <chr>    <dbl>
## 1 2014  10369593
## 2 2015  10297138
## 3 2016  10204057
## 4 2017  10124579
## 5 2018  10049607

연산자 %>% 는 연결 연산자(pipe operator)라고 하며 함수를 축차적으로 연속하여 사용하지 않고 왼쪽에서 생성된 자료와 정보를 오른쪽 함수의 인자로 전달 해주는 역활을 하는 유용한 연산자이다. 위의 명령어는 3개의 단계로 이루어 진다.

  • 데이터프레임 seoulpop 지정한다 (지정한 후에는 변수이름을 자유롭게 이용할 수 있다)
  • 변수 year를 함수 group_by로서 그룹변수로 지정한다
  • 새로운 변수 totalpop는 인구 pop를 함수 sum을 이용하여 합계를 구한다 (함수의 작업은 앞에서 지정된 그룹별로 이루어 진다).

위의 요약된 정보를 yeartotal에 저장하려면 다음과 같은 명령어를 사용하면 된다.

yeartotal <- seoulpop %>% group_by(year) %>% summarise(totalpop = sum(pop))
