문제 1.

나이(age)와 임금(logwage)에 대한 산포도를 그리시오.

문제 2.

나이(age)의 구간을 \((21,65)\)으로 고려하자. 구간을 나누는 개수를 \(k=5,10,15,20\)으로 하여 임금(logwage)에 대한 regressogram을 그리시오.

문제 3.

나이(age)를 설명변수로 하여 임금(logwage)의 평균 함수를 정규분포 확률밀도함수를 이용한 커널 추정법으로 추정하려고 한다. CV를 최소로 하는 최적 구간의 길이 \(h\) 를 구하시오.

문제 4.

문제 3 에서 구한 최적 구간의 길이 \(h\)를 이용하여 임금(logwage)의 커널 추정 함수와 95% 신뢰 구간을 그리시오.

문제 5.

앞에서 분석한 결과를 가지고 나이와 임금의 관계에 대하여 간략하게 설명하시오.

regressogram = function(x,y,left,right,k,plotit,xlab="",ylab="",sub=""){
  ### assumes the data are on the interval [left,right]
  n = length(x)
  B = seq(left,right,length=k+1)
  WhichBin = findInterval(x,B)
  N = tabulate(WhichBin)
  m.hat = rep(0,k)
  for(j in 1:k){
    if(N[j]>0)m.hat[j] = mean(y[WhichBin == j])
  }
  if(plotit==TRUE){
    a = min(c(y,m.hat))
    b = max(c(y,m.hat))
    plot(B,c(m.hat,m.hat[k]),lwd=3,type="s",
         xlab=xlab,ylab=ylab,ylim=c(a,b),col="blue",sub=sub)
    points(x,y)
  }
  return(list(bins=B,m.hat=m.hat))
}

kernel = function(x,y,grid,h){
  ### kernel regression estimator at a grid of values
  n = length(x)
  k = length(grid)
  m.hat = rep(0,k)
  for(i in 1:k){
    w = dnorm(grid[i],x,h)
    m.hat[i] = sum(y*w)/sum(w)
  }
  return(m.hat)
}
kernel.fitted = function(x,y,h){
  ### fitted values and diaginal of smoothing matrix
  n = length(x)
  m.hat = rep(0,n)
  S = rep(0,n)
  for(i in 1:n){
    w = dnorm(x[i],x,h)
    w = w/sum(w)
    m.hat[i] = sum(y*w)
    S[i] = w[i]
  }
  return(list(fitted=m.hat,S=S))
}


h=0.8
