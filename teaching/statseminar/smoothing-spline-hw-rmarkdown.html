문제 1.

나이(age)와 임금(logwage)에 대한 산포도를 그리시오.

문제 2.

나이(age)의 구간을 \((21,65)\)으로 고려하자. 구간을 나누는 개수를 \(k=5,10,15,20\)으로 하여 임금(logwage)에 대한 자연 3차 스플라인(natural cubic spline) 함수추정식을 그리시오.

library(splines)
kknots=seq(21+0.1,65-0.1,len=5)
x.grid = seq(21,65,length.out = 100)
age = cps71$age
logwage = cps71$logwage
fit1 = lm(logwage ~ ns(age,knots = kknots ))
pred = predict(fit1,newdata=data.frame(age=x.grid))
plot(age,logwage,pch=20)
lines(x.grid, pred, col="red")
abline(v=kknots, col="blue")

문제 3.

나이(age)를 설명변수로 하여 임금(logwage)의 평균 함수를 평활 스플라인(smoothing spline)
을 유효 자유도를 \(df = 5,10,15,20\)로 하여 추정하고 추정함수식릏 그리시오.

age = cps71$age
logwage = cps71$logwage
fit2 = smooth.spline(logwage  ~ age, all.knots=T, cv=T, df=10)
## Warning in smooth.spline(logwage ~ age, all.knots = T, cv = T, df = 10):
## cross-validation with non-unique 'x' values seems doubtful
plot(age,logwage,pch=20)
lines(fit2,col="red")

문제 4.

다음은 R package 의 nlstools에 있는 O2K자료이다. 폐질환이 있는 환자가 처음 5.83분까지는 안정을 취하다가 이후 6분을 걸으면서 산소 흡수량(VO2, Oxygen uptake kinetics)을 시간 t별로 측정한 값이다.

시간에 따른 산소 흡수량의 변화를 평활스플라인으로 적합하고 추정된 함수식을 그림을 그리시오.

문제 5

위 문제 4에서 사용한 산소흡수량에 대한 자료를 커널회귀모형을 이용하여 적합해 보시오. R 에서 커널회귀모형은 함수 ksmooth를 사용한다. 다음은 정규분포함수(normal)를 커널로 사용하고 구간의 길이를 4로 사용한 경우(bandwidth = 4)이다.

구간의 길이를 몇 가지로 선택하여 변화시키면서 가장 적합한 길이를 찾아보시오.

plot(x,y,pch=20)
fit4 <- ksmooth(x, y, "normal", bandwidth = 4)
lines(fit4, col = 2)
